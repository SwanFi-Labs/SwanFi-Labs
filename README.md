# SwanFi Labs

> Building BSC token discovery & trading infrastructure

SwanFi is a real-time BSC token discovery and trading platform — tracking new token launches, bonding curve progress, migrations, and on-chain activity across Flap.sh and Four.meme.

## Features

- Real-time token discovery (new, migrating, migrated)
- Live bonding curve progress tracking
- On-chain transaction & holder analytics
- WebSocket-driven price and trade feeds
- Public REST API with rate limiting
- Multi-provider BSC RPC with circuit breaker
